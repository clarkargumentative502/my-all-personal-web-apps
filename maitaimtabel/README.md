# 📅 Maitaimtabel

Maitaimtabel adalah aplikasi jadwal harian berbasis Single Page Application (SPA) yang ringan, offline-friendly, dan tanpa dependency eksternal.

## ✨ Fitur Utama

- ✅ Dashboard jadwal harian
- 📆 Navigasi waktu (Kemarin, Hari Ini, Besok)
- 🏷️ Kategori dinamis
- 🔍 Sorting & filtering otomatis
- 📋 Copy jadwal ke clipboard
- 💾 Export / Import JSON
- ⚡ 100% Vanilla JS (tanpa framework)

## 🚀 Teknologi

- HTML5
- CSS3 (Mobile-first)
- Vanilla JavaScript
- localStorage API
- Clipboard API
- File API

## 📱 Design Principles

- Mobile-first
- Minimalis & clean
- Warna pastel (eye-friendly)
- Tanpa CDN / library eksternal

## 📦 Cara Menjalankan

1. Download file `index.html`
2. Buka di browser
3. Langsung bisa digunakan (tanpa server)

## 💾 Penyimpanan Data

Semua data disimpan di:

localStorage browser

Tidak ada backend / server.

## 🔄 Backup Data

### Export
- Klik tombol "Export"
- File `.json` akan terdownload

### Import
- Klik "Import"
- Pilih file JSON
- Data akan replace data lama

## 📋 Share Jadwal

- Klik "Copy Hari Ini"
- Paste ke WhatsApp / Notes

## ⚠️ Catatan

- Data tidak sinkron antar device
- Jika clear browser storage → data hilang
- Disarankan rutin export backup
