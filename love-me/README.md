# 💝 Love Me: Pesan Cinta untuk Dirimu

Sebuah web app sederhana untuk membantu kamu berhenti sejenak, bernapas, dan mengatakan sesuatu yang baik kepada dirimu sendiri.

Dibangun sebagai pengalaman ringan tanpa backend, berjalan sepenuhnya di browser menggunakan HTML, CSS, dan JavaScript murni.

---

## 💓 Fitur

### 💖 Mode Hening (Ruang Bernapas)
- Ketuk orb untuk masuk ke mode tenang
- Suara ambient lembut menggunakan Web Audio API
- Panduan napas: *Tarik Napas... / Hembuskan...*
- Perubahan background yang halus untuk menciptakan suasana berbeda

---

### 🧡 Jurnal Self-Compassion
- Tulis hal baik tentang dirimu hari ini
- Simpan secara lokal (tanpa akun, tanpa server)
- Mendapatkan pesan afirmasi setelah menyimpan

---

### 💝 Diary Hati
- Melihat kembali catatan yang pernah kamu tulis
- Hapus per item
- Hapus semua catatan jika diperlukan

---

### 💞 Micro Interaction
- Animasi orb yang “bernapas”
- Transisi UI yang lembut
- Pesan feedback yang muncul dan menghilang perlahan
- Respon visual kecil saat menyimpan

---

## ❤️ Filosofi

Aplikasi ini bukan tentang produktivitas.

Tapi tentang:
- berhenti sejenak
- mengakui usaha diri sendiri
- memberi ruang aman untuk diri sendiri

Tanpa tekanan.  
Tanpa penilaian.  
Hanya ruang kecil untuk bernapas.

---

## 🛠️ Teknologi

- HTML5
- CSS3 (custom properties & transition)
- JavaScript (Vanilla)
- Web Audio API (suara ambient)
- Canvas API (background partikel)
- LocalStorage (penyimpanan data)

---

## 📱 Cara Menggunakan

Cukup buka file `index.html` di browser.

Tidak perlu instalasi.

Lebih nyaman digunakan di perangkat mobile.
