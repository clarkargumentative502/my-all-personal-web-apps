# Simpel Kalender Indonesia

Simpel Kalender Indonesia adalah aplikasi web kalender modern, ringan, dan responsif yang dirancang khusus untuk pengguna di Indonesia. Aplikasi ini menggabungkan kesederhanaan kalender Masehi dengan data hari libur nasional Indonesia yang akurat.

## ✨ Fitur Utama

- **Real-time Today View**: Menampilkan tanggal hari ini dengan tipografi yang besar dan jelas.
- **Hari Libur Nasional Indonesia**: Integrasi otomatis dengan data hari libur nasional (tanggal merah) dan peringatan penting di Indonesia.
- **Navigasi Intuitif**: Dukungan navigasi antar bulan/tahun menggunakan tombol atau gesture *swipe* pada perangkat layar sentuh.
- **Tombol "Hari Ini"**: Akses cepat (Floating Action Button) untuk kembali ke tanggal hari ini dari posisi navigasi mana pun.
- **Copy to Clipboard**: Cukup ketuk header tanggal untuk menyalin format tanggal lengkap ke papan klip.
- **Daftar Hari Penting**: Ringkasan otomatis hari-hari libur yang terjadi pada bulan yang sedang dilihat.
- **Desain Modern**: Menggunakan antarmuka bersih yang nyaman di mata (Mobile-first).

## 🛠 Teknologi yang Digunakan

- **HTML5**: Struktur semantik.
- **CSS3**: Styling utilitas untuk desain yang responsif dan estetis.
- **Vanilla JavaScript**: Logika aplikasi tanpa ketergantungan framework berat.
- **Holidays ID API**: Sumber data hari libur nasional Indonesia berbasis JSON dari GitHub.

## 🚀 Cara Penggunaan

Aplikasi ini bersifat *portable* dan tidak memerlukan instalasi server yang kompleks.

1. Simpan kode HTML ke dalam file bernama `index.html`.
2. Buka file tersebut menggunakan browser modern (Chrome, Safari, Edge, atau Firefox).
3. Pastikan perangkat terhubung ke internet untuk pertama kali agar aplikasi dapat mengambil data hari libur nasional.

## 🖥️ Sumber Data API

Aplikasi ini mengambil data statis dari repositori GitHub

## 💾 Penyimpanan & Performa

- **Tanpa Database**: Aplikasi berjalan sepenuhnya di sisi klien (*client-side*).
- **Caching**: Data libur diambil secara dinamis setiap kali terjadi perpindahan tahun untuk memastikan akurasi data.
