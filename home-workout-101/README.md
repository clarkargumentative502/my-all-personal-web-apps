# 💪 Home Workout 101

Home Workout 101 adalah web app sederhana untuk mencatat dan menjalankan workout harian langsung di browser.

Dirancang dengan konsep **minimal, terstruktur, dan offline-friendly**—seperti mengelola project, tapi untuk tubuhmu sendiri.

---

## ✨ Fitur Utama

- 📅 Routine harian (Upper, Lower, Full Body, Custom)
- 💪 Tracking set & progress per exercise
- 📊 Progress summary (exercise, set, percentage)
- ⏱️ Rest timer dengan countdown & sound
- 🌙 Light / Dark mode
- 💾 Auto-save ke localStorage
- 📱 Mobile-first & responsive
- ⚡ 100% Vanilla JS (tanpa framework)

---

## 🧠 Cara Kerja

Semua data dan interaksi berjalan langsung di browser.

- Tidak ada backend
- Tidak ada koneksi server
- Semua progress tersimpan otomatis di device

---

## 🛠️ Teknologi

- HTML5
- CSS3 (Flexbox, Grid, Custom Properties)
- Vanilla JavaScript (ES6+)
- Web APIs:
  - localStorage
  - Web Audio API

---

## 📱 Cara Penggunaan

1. Pilih routine (Monday / Wednesday / Friday / Custom)
2. Selesaikan set dengan klik checklist
3. Gunakan rest timer jika diperlukan
4. Lihat progress di summary
5. Selesaikan workout

Semua progress tersimpan otomatis.

---

## 💾 Penyimpanan Data

Menggunakan `localStorage` browser untuk:

- Progress workout
- Routine
- Preferensi (theme)
- History sederhana

---

## ⚠️ Catatan

- Data tidak sinkron antar device
- Jika storage browser dihapus → data hilang
- Disarankan backup manual (fitur export di future)

---

## 🎯 Future Improvement

## 📦 Cara Menjalankan

1. Download semua file
2. Buka `home-workout-101.html` di browser
3. Langsung bisa digunakan

Tidak perlu install, build, atau server.
